A Boltzmann Informed Model for Designing Cyclic Peptides via Diverse Chemistries (MPhil Project)

The goal of this project is to use a machine learning model, which can sample from a peptide's conformational Boltzmann distribution, to suggest the best ways to then cyclize that peptide via at run time conditioning.
